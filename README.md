## 📚 FeedStack – Institution Feedback Management System
FeedStack is a simple feedback collection web app built for Vidyarthi Education.  
Students can rate subjects, evaluate individual teachers, and share suggestions through a single form.

This project focuses mainly on strengthening my HTML fundamentals and understanding how form-based systems work.

## 🚀 Features
- Add and evaluate multiple teacher(s) dynamically
- Mobile based responsive design
- Firebase Firestore database storage   
- Subject-wise ratings  
- Simplistic study environment feedback  



## 🧠 What I Learned
- Advanced HTML form structuring  
- DOM manipulation  
- Dynamic element cloning  
- Basic Firebase Firestore integration  
- Mobile-friendly UI design  

## 🛠 Built Using
- HTML  
- CSS  
- JavaScript  
- Firebase Firestore  
- GitHub Pages  

## 📌 Purpose
Built to help teacher's collect digital feedback.
